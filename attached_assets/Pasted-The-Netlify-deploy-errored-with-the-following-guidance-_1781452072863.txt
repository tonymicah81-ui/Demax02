Diagnosis
- Error type: runtime/build-time command not found.
- Cause: the Vite CLI binary is not available in the build environment, so the npm script "vite build" cannot be executed. This commonly happens when Vite is listed only in devDependencies and Netlify's install step did not install devDependencies (or Vite is missing from package.json entirely).

Required verification (do this first)
- Verify vite is declared in your package.json (either dependencies or devDependencies): https://github.com/tonymicah81-ui/Demax02/blob/main/package.json

Solutions (pick one)

1) Preferred — Install devDependencies during Netlify build
- Configure Netlify to install devDependencies so the Vite binary is present. Add (or update) netlify.toml with an environment flag that tells npm to include dev deps:

```toml
# netlify.toml
[build]
  command = "npm run build"
  publish = "dist"

[build.environment]
  NPM_FLAGS = "--include=dev"
```

- Commit that change and re-deploy. Alternatively, set the environment variable NPM_FLAGS = --include=dev in the Netlify UI (Site settings → Build & deploy → Environment → Edit variables).

2) Alternative — Move Vite into production dependencies
- If you prefer not to install devDependencies in CI, move Vite from devDependencies to dependencies so it is installed in production installs:
  - Run locally: npm install vite --save
  - Commit package.json and package-lock.json and push. This ensures vite is present during Netlify's install step.

3) Quick workaround — Use a local-installed binary via npm exec
- If vite is present in devDependencies and you want to avoid changing Netlify config, change build script to explicitly call npm exec which will use a project-installed binary only if it exists (but note this still requires devDependencies to be installed):
  - package.json "scripts": { "build": "npm exec -- vite build" }

Notes
- If after making the above change you still see the npm internal error from [line 17](#L17)-[line 20](#L20), try clearing Netlify's cache and re-deploy. If Node version issues arise, follow Netlify docs for changing Node versions: https://docs.netlify.com/configure-builds/manage-dependencies/#node-js-and-javascript

Summary
- The build fails because the "vite" CLI is not available during the build ([line 62](#L62)). Verify vite is in package.json, then either make Netlify install devDependencies (set NPM_FLAGS="--include=dev") or move vite into dependencies so the binary is installed during the build.
